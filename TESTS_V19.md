# فحوص V19

## البناء

- `npm run build`: نجح TypeScript وVite وPWA بدون أخطاء.
- حجم JavaScript المضغوط تقريبًا: 147 كيلوبايت gzip.
- Service Worker تم إنشاؤه وضم ملفات التطبيق الأساسية إلى precache.

## اختبارات منطقية آلية

تم تنفيذ Assertions فعلية للسيناريوهات التالية:

1. يوم عادي مع 7 ساعات بدون أكل، مياه قليلة، نوم 5 ساعات وجيم قريب:
   - إنشاء مهمة أكل عاجلة.
   - إنشاء مهمة مياه.
   - إنشاء مراجعة طاقة بسبب قلة النوم.

2. نفس الحالة أثناء الصيام:
   - عدم إنشاء مهمة أكل.
   - عدم إنشاء مهمة مياه.
   - عدم إنشاء تذكير كرياتين أثناء الصيام.
   - عدم إصدار تنبيه «كل الآن» أو «اشرب الآن».
   - رد المدرب المحلي يراعي الصيام.

3. مهمة وجبة قبل العصر بخمس دقائق:
   - تم نقلها لما بعد الصلاة تلقائيًا.

4. البحث العربي:
   - «طعميه» يطابق «طعمية».
   - «ارز» يطابق «أرز أبيض».

5. مواقيت الصلاة المحلية للقاهرة بتاريخ 12 يوليو 2026:
   - الفجر 04:19.
   - الظهر 13:02.
   - العصر 16:37.
   - المغرب 20:00.
   - العشاء 21:30.

هذه القيم فحص منطقي للحساب المحلي وليست بديلًا عن مواقيت الجهة الرسمية؛ قد توجد فروق دقائق بسيطة.

## ملاحظة اختبار الواجهة

تمت مراجعة JSX وCSS والبناء الإنتاجي واستجابة الخادم. بيئة التشغيل الآلي المتاحة منعت Chromium من فتح عناوين محلية بسياسة إدارية، لذلك يظل اختبار اللمس الفعلي على هاتف أندرويد خطوة موصى بها قبل النشر النهائي، خصوصًا للإشعارات وتثبيت PWA.
