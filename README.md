# GYM Life Coach — V19 Stability & UX Audit

مساعد يومي عربي يعيد ترتيب اليوم حسب ما يحدث فعلًا، ويعمل بمحرك قواعد محلي مجاني، مع مدرب AI اختياري.

## الجديد في V19

- مراجعة كاملة لمنطق التطبيق وتجربة الموبايل.
- إصلاح وضع رمضان ومنع اقتراح الأكل والمياه أثناء الصيام.
- تمرير كل مهام المحرك الجديدة على أولوية الصلاة.
- حساب محلي لمواقيت الصلاة عند غياب الإنترنت.
- إصلاح المياه والتقدم والوزن وحذف الطعام المخصص.
- واجهة يوم أخف: الخطوة المهمة أولًا والتفاصيل قابلة للفتح.
- زر موبايل ثابت «إيه اللي حصل؟» وحفظ ثابت للإعدادات.
- Error Boundary، تحقق من بيانات المستخدم، وتحسين إشعارات PWA.

راجع `CHANGES_V19.md` و`TESTS_V19.md` و`REVIEW_V19.md` للتفاصيل.

## أساس V18

### 1) Smart Engine محلي — مجاني وبدون إنترنت

- يعرف وقت الجهاز الحالي ويتابع تغيّر اليوم تلقائيًا.
- يعرف الصيف والشتاء ويستخدم الموسم في اختيارات الطعام والمشتريات.
- يفعّل منطق الجمعة ويعطي الصلاة الأولوية.
- يعرف هل المستخدم في البيت أو خارج البيت أو في الجيم.
- يقرأ الوجبات المسجلة، ساعات النوم، المياه، الكرياتين، الطاقة، التعب والهدف.
- يعيد بناء المهام تلقائيًا كل عدة دقائق وعند تسجيل أي حدث.
- يضيف خطوة أكل عاجلة بعد فجوة طويلة بدون أكل.
- يخفف الجيم عند قلة النوم أو انخفاض الطاقة.
- يضيف تعويض مياه تدريجيًا حسب وقت اليوم.
- يحرّك تذكير الكرياتين لو تأخر بدون ربطه بالصباح فقط.

### 2) «اسأل المدرب الذكي»

- محادثة طبيعية داخل التطبيق.
- أمثلة جاهزة: التعب، ميزانية الطعام، الطعام المتاح، قرار الجيم، ونسيان الكرياتين.
- رد محلي مجاني يعمل دائمًا بدون إنترنت.
- إمكانية ربط API خارجي متوافق بشكل اختياري من الإعدادات.
- رجوع تلقائي للرد المحلي لو فشل الاتصال بالـAI.
- سجل محادثة يومي محفوظ محليًا في IndexedDB.

### 3) المدرب يبدأ الكلام

- يلاحظ مرور أكثر من 5 ساعات بدون أكل.
- ينبه عند انخفاض المياه في وقت متأخر من اليوم.
- يضع الصلاة قبل الجيم عند قرب الموعد.
- يقترح تقليل شدة الجيم عند قلة النوم.
- يذكّر بالكرياتين في الوقت المناسب.
- يظهر الكلام داخل التطبيق، مع إشعارات جهاز اختيارية عند منح الإذن.
- يمكن إخفاء كل ملاحظة بدون أن تتكرر فورًا.

## مزايا V17 المحفوظة

- الصلاة جزء من «مدير يومك» وليست كارت معلومات فقط.
- زر موحد «إيه اللي حصل فعلًا؟».
- تقييم الطاقة ووضع «أنا تعبان».
- قائمة مشتريات ذكية وتحليل كفاية الوجبة.
- تسجيل المطاعم والوجبات المعتادة.
- وضع الجمعة ووضع رمضان.
- النوم والاستيقاظ الحقيقي، المياه، الكرياتين، الجيم، الوزن والتقدم.
- PWA وتخزين محلي باستخدام Dexie / IndexedDB.

## ملاحظة أمان للـAI الاختياري

مفتاح API في تطبيق ويب يمكن الاطلاع عليه من الجهاز. للاستخدام العام أو الإنتاجي، الأفضل وضع الاتصال بمزود الـAI خلف Proxy أو Cloudflare Worker خاص بك بدل تضمين المفتاح في الواجهة.
