# GYM Life Coach

مساعد يومي عربي للرجال فقط، يربط:
- النوم والاستيقاظ الحقيقي
- الأكل المتاح بنظام متاح / غير متاح
- المياه
- الكرياتين
- رحلة الجيم: رايح الجيم → أنا في الجيم → خلصت الجيم
- حالة خارج البيت مع اقتراحات سوبر ماركت
- متابعة التقدم والوزن
- تسجيل اختياري للعادة السرية بدون سؤال يومي
- بيانات التدخين داخل التوجيهات

## التشغيل

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```

## النشر

الملف `wrangler.jsonc` موجود لتحديث Worker الحالي `web-gym`.
