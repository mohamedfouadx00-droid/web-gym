# GYM Daily Manager

مساعد يومي عربي للرجال فقط، يركز على تنظيم الأكل والمياه والكرياتين والجيم والنوم حسب واقع اليوم.

## أهم الوظائف
- زر «أنا صحيت الآن»
- «ماذا أفعل الآن؟»
- «أنا رايح الجيم الآن»
- «رجعت من الجيم»
- «يومي اتلخبط»
- «أنا بره البيت»
- تم / تأجيل / غير متاح لكل خطوة
- إعادة ترتيب اليوم تلقائيًا
- مخزون أكل بالكميات
- تقييم نهاية اليوم
- مراجعة أسبوعية
- اقتراح تعديل السعرات من اتجاه الوزن والالتزام

## التشغيل
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

## النشر
ارفع الملفات إلى GitHub. الـ GitHub Action الموجود يبني المشروع، وCloudflare يعيد النشر تلقائيًا إذا كان الريبو مربوطًا به.
