# تغييرات V18 — Smart Engine & Optional AI Coach

## المحرك المحلي
- إضافة `smartEngineService.ts` لتكوين لقطة كاملة لحالة اليوم.
- قواعد تلقائية للنوم والأكل والمياه والكرياتين والمكان والجمعة.
- تحديث دوري لخطة اليوم كل خمس دقائق أثناء فتح التطبيق.
- بقاء كل القرارات الأساسية محلية وبدون AI أو إنترنت.

## المدرب الذكي
- إضافة `smartCoachService.ts` بمحرك فهم محلي للأسئلة العربية الشائعة.
- دعم API خارجي متوافق مع Chat Completions بشكل اختياري.
- Fallback محلي عند انقطاع الإنترنت أو فشل الـAPI.
- تخزين رسائل المحادثة اليومية في جدول `coachMessages`.

## المبادرة التلقائية
- رسائل استباقية عند فجوة أكل طويلة، نقص المياه، قلة النوم، قرب الصلاة من الجيم، تأخر الكرياتين، أو الخروج من المنزل.
- إشعارات متصفح اختيارية.
- حفظ التنبيهات المخفية لمنع الدوران والتكرار الفوري.
